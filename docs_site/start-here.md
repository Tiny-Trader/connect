# Start Here

Pick your path.

## Path A: Beginner (first successful order)
1. Read [Basics](basics.md)
2. Configure [Login & Session](login-and-session.md)
3. Verify keys in [Config & Environment](config-and-env.md)
4. Run [Recipe: First order](recipes/first-order.md)
5. If blocked, use [Troubleshooting](troubleshooting/index.md)
