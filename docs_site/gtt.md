# GTT (Trigger Orders)

## What GTT means
A rule that triggers an order when price condition is met.

## Common actions
- create trigger rule
- modify trigger rule
- cancel trigger rule
- list/fetch rules

## Rule shapes
- single leg trigger
- OCO (one-cancels-other) where broker supports it

## Practical notes
- GTT behavior is broker-specific
- always confirm trigger status after creation
